BLUEPRINT GRAMMAR


BLUEPRINT (<ROW>\n?)*
ROW \| <PIXELS>? (<ITEM><PIXELS>?)* <PIXELS>? \|
PIXELS \d*|\*
ITEM [ <IDENT> <HSIZE>? <VSIZE>? ]
IDENT [a-zA-Z_$][0-9a-zA-Z_$]*
HSIZE PIXELS|\d*-PIXELS
VSIZE PIXELS|\d*|PIXELS


EXAMPLES

      | 15 [ IDENT0 50-75 25 ] 5 [ IDENT2 150-250 25      ] 15 | 
      | 15 [ IDENT3 50-75 16 ] 5 [ IDENT4 150-250 100|300 ] 15 | 

      | * [ IDENT0 50-75 25 ] 5 [ IDENT2 150-* 25    ] * |
      | * [ IDENT3 50-75 16 ] 5 [ IDENT4 150-* 100|* ] * |
